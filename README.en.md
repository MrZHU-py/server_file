# server_file

#### Description
基于多线程实现的服务器文件上传，使用了线程池，支持多用户同时上传文件。基于 SHA256 和 Redis 数据库，实现“文件秒传”功能，避免了资源浪费，同时能够记录上传过的文件信息并保存到日志中。
